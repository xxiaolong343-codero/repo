# MISRA-C 2012 关键规则速查表

## 1. 类型与表达式

| 规则号 | 内容 | 检查项 | 合规要求 |
|--------|------|--------|----------|
| **Dir 4.6** | 使用 typedef 表明类型意义 | 是否用裸 int/unsigned？ | 使用 stdint.h 类型 |
| **Rule 10.1** | 操作数类型不应隐式转换 | 是否有隐式转换？ | 必须显式 cast |
| **Rule 10.4** | 算术运算两端类型相同 | 两端类型是否一致？ | 相同类型运算 |
| **Rule 12.2** | 移位操作右值在范围内 | 移位位数是否合法？ | 0 ≤ 右值 < 左值位数 |

## 7. 豁免记录模板

当必须豁免某条规则时，在代码中添加注释：

```c
/* MISRA-C 豁免: Rule 15.1 - 使用 goto 跳出多层循环
 * 理由: 该场景使用标志变量会导致代码复杂度显著增加
 * 审查者: 张三
 * 日期: 2026-06-17
 */
goto cleanup;
```

## 8. 合规级别说明

| 级别 | 含义 | 处理方式 |
|------|------|----------|
| **强制** | 必须遵守，不得豁免 | Code Review 必查 |
| **必需** | 应遵守，可经评审后豁免 | 需记录豁免理由 |
| **建议** | 推荐遵守 | 最佳实践 |
