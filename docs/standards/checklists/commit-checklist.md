# 提交前自检清单

每次 `git commit` 前检查：

---

## 编译检查

- [ ] RL78 平台编译通过（CS+）
- [ ] HDSC 平台编译通过（Keil MDK）
- [ ] 无编译警告（或已知的可接受警告）

---

## 测试检查

- [ ] 单元测试通过（如适用）
- [ ] 新增代码有对应单元测试

---

## MISRA-C 检查

- [ ] 无强制规则违规
- [ ] 必需规则如有豁免，已添加豁免注释

---

## 代码规范检查

- [ ] 提交消息符合格式（`feat/fix/docs...`）
- [ ] 变更行数 < 500 行（大变更拆分多次提交）
- [ ] 函数命名符合规范
- [ ] 公开 API 有注释说明

---

## 文档检查

- [ ] 是否需要更新设计文档？
- [ ] 是否需要更新 README？

---

## 最终确认

- [ ] 我确认以上所有项已检查通过
